コミュニケーションの文脈

どこまで言葉で明示するかに関する価値観です。

ローコンテクスト

  • 言葉にされた情報を重視します。
  • 前提・結論・背景を明確に共有することで、誤解を防げると考えます。

ハイコンテクスト

  • 言葉以外の背景や空気、関係性を重視します。
  • 文脈や行間を読み取ることが自然だと感じます。
{% elif key == "feedback" %}

相手への批判の伝え方

批判をどう伝えるかに関する価値観です。

直接的なネガティブ・フィードバック

  • 問題点や批判は率直に伝える方が誠実だと考えます。
  • 指摘と人格は別物だという前提があります。

間接的なネガティブ・フィードバック

  • 相手の感情や関係性への配慮を重視します。
  • 柔らかい表現や前置きを通じて伝えることが自然です。
{% elif key == "persuasion" %}

相手を説得する方法

「なぜ納得するか」「どう説明されると腹落ちするか」の価値観です。

原理優先（なぜ？）

  • 背景・理由・考え方を理解した上で納得したいタイプです。
  • 筋が通っていることを重視します。

応用優先（どうやって？）

  • 具体的な方法や手順が示されると納得しやすいタイプです。
  • 実行イメージが持てることを重視します。
{% elif key == "hierarchy" %}

上司と部下の関係

組織内の立場や役割をどう捉えるかの価値観です。

平等主義的

  • 上司と部下は対等に意見を出し合う関係だと考えます。
  • フラットなコミュニケーションを好みます。

階層主義的

  • 役職や立場には明確な意味があると考えます。
  • 序列に沿った指示や報告が重要だと感じます。
{% elif key == "decision" %}

意思決定の方法

どのように決めると納得できるかの価値観です。

合意志向

  • 関係者の意見をすり合わせ、納得感を重視します。
  • 時間がかかっても合意形成を大切にします。

トップダウン式

  • 決定権を明確にし、スピードと実行力を重視します。
  • 最終判断はリーダーが行うべきだと考えます。
{% elif key == "trust" %}

信頼する方法

何をもって相手を信頼するかに関する価値観です。

タスクベース

  • 仕事の成果や約束を守ることによって信頼が築かれます。
  • 実務上の信頼を重視します。

関係ベース

  • 一緒に過ごす時間や個人的な関係性を通じて信頼が育ちます。
  • 人としてのつながりを重視します。
{% elif key == "conflict" %}

見解の相違があった場合

意見の違いにどう向き合うかの価値観です。

対立型

  • 議論や反論は、より良い結論のために必要だと考えます。
  • 公の場での意見のぶつかり合いも問題ないと感じます。

対立回避型

  • 対立は関係性に影響すると感じます。
  • 意見の違いは慎重に、穏やかに扱いたいと考えます。
{% elif key == "time" %}

効率的なスケジューリングの方法

時間や計画をどう扱うかに関する価値観です。

直線的な時間

  • 計画通りに一つずつ進めることを重視します。
  • 締切やスケジュールの厳守が重要です。

柔軟な時間

  • 状況に応じて予定を変えることは自然だと考えます。
  • 優先順位の変化や割り込みにも対応します。
