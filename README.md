## アプリ名　　Teamy
個人の価値観を可視化して、コミュニケーションの最適化を図る

### コミュニケーションをもっとスムーズに
Teamyは自分と相手(チーム)の「価値観の違い」を
見える化しコミュニケーションの最適解を教えてくれるアプリです

### 価値観はさまざま、コミュニケーションはひとつだけ？
人は誰でも自分なりの「正しいコミュニケーションの型」を持っています。
ただしそれはチームや相手が変われば正解でなくなることがあります。
Teamyが重視するのは「相対性」です。

### 違いを知るとあなたの言動が変わるかも
自分の価値観と相手の価値観、その差はどのくらいあるのか
それによって「そのままがいい」「少し調整する」「意識的に変える」など
あなたの行動指針をTeamyがアドバイスします
